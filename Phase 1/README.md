# Todo Application – Phase 1

A command-line based Todo application using Python that stores all tasks in memory.

## Features

- Add new tasks (title only)
- View all tasks with their status
- Update tasks (mark complete/incomplete)
- Delete tasks

> ⚠️ Phase-1 is **console-only** and **in-memory**. There is no database, web UI, or persistence.

## Prerequisites

- Python 3.10 or higher (Python 3.13 also works)
